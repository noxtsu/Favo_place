# Favo_place


## サイト概要
自分が好きな場所、思い出の場所を記録＆シェアできるサイト

### サイトテーマ
思い出の場所、紹介したい場所を画像と文章で記録。

### テーマを選んだ理由
<私は日本中のいろんな地域に行って、そこにどんな風景があるのかを肌で感じてみたいと思っています。
地図アプリはGoogleマップで位置を調べられます。写真の記録はインスタグラム等で記録、発信できます。
しかしこのアプリは場所を起点に風景などを写真で記録することに特化したものです。
地元にあった小さな由緒ある神社、福岡で食べた屋台のラーメン、初めて東京で一人暮らしした葛飾柴又…
思い出に残っている記憶は「どこ」に紐づいていると感じます。
そんな記憶を残せるサイトがあればと思い、このテーマにしました。

このサイトでできること
・自分の思い出の場所を記録できる。
・自分の興味がある場所に関して、他のユーザのものを参考に出来る。
・シェアされた投稿を見て初めて訪れる土地に何があるのかがわかる。
・全年齢対象のため、操作性良く場所と記録を直感的に記録できる。

### ターゲットユーザ
全年齢：国内旅行が好きな方。身近な場所のPRをしたい方

### 主な利用シーン
旅行の思い出を残したいとき、心に残った風景を記録したいとき
