# Favo_place


## サイト概要
自分が好きな場所、思い出の場所を記録＆シェアできるSNSサイト
地図機能を使い、特定の場所に関して自分の思い出等をブログ風につづり投稿。
サイト利用者にシェアできます。
ユーザーは特定の場所に関しての情報や、
その場所にまつわる他ユーザーエピソードを閲覧できるほか、
特定のユーザーの投稿を閲覧できる機能もあります。
その場所に関しての思いを気軽にブログ風に紹介できる、
その投稿を読んでどんな場所かを知ることができる
場所にスポットを当てたSNSサイトです。

このサイトでできること
・自分の思い出の場所を記録できる。
・自分の興味がある場所に関して、その土地がどういう場所なのかがわかる。
・面白いと思ったユーザーの投稿を追ってみることができる（フォロー機能）


### サイトテーマ
自分にとっての特別な場所を発信、その場所の思い出シェアサイト

### テーマを選んだ理由
私は日本中のいろんな地域に行って、そこにどんな風景があるのかを肌で感じてみたいと思っています。
地元にあった小さな由緒ある神社、福岡で食べた屋台のラーメン、初めて東京で一人暮らしした葛飾柴又。
思い出に残っている記憶は「どこ」に紐づいていると感じます。

現代ではTwitter、Instagram、Googleマップ等でレビューとして場所の情報があふれていますが、
短文ではなくしっかり文章化して思いを綴ることは少なくなってきたのではと感じます。

場所の思い出をしっかり残せる、そしてそれを発信できる。
そんな思いを残せるサイトがあればと思い、このテーマにしました。

### ターゲットユーザ
その場所に関して思いを綴りたい方（昔住んでいた町を紹介したい、旅行で訪れた場所の思い出を残したい）
その場所を知りたいと思う方（旅行先を考える方。新居をどこにしようか考えている方）
気軽にブログを書きたい方。


### 主な利用シーン
旅行の思い出を残したいとき、心に残った風景を記録したいとき
